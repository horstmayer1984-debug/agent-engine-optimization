# KI-Empfehlungszuordnung für AEO SEO

Die KI-Empfehlungszuordnung für AEO SEO verbindet drei Signale: Suchimpressionen, KI-gestützte Entdeckung und Aktionen zur Agentengewinnung. Die Search Console zeigt die Abfragenachfrage und die Seitenleistung an. Analytics kann KI-Referenzen anzeigen. Protokolle der Ausführungsschicht zeigen, ob Agenten tatsächlich Aufgaben abgeschlossen haben.

## Warum die Search Console nur die erste Ebene ist

Der Leistungsbericht der Search Console ist nützlich, da er Suchanfragen, Seiten, Impressionen, Klicks, CTR und durchschnittliche Position anzeigt. Wenn eine Suchanfrage wie „aeo seo“ zunimmt, bedeutet dies, dass die Nachfrage steigt.

Aber die Search Console zeigt nicht alle KI-Touren an:

- Eine KI-Antwort kann eine Seite erwähnen, ohne darauf zu klicken
- Ein Chatbot kann Empfehlungsverkehr senden
– Ein Agent kann einen Endpunkt anrufen oder einen Warenkorb erstellen
- Ein Benutzer kann später über den direkten Verkehr zurückkehren

Aus diesem Grund benötigt die AEO-SEO-Messung mehr als einen Bericht.

## Messebenen

| Schicht | Tool oder Datenquelle | Welche Antworten |
|
---|
---|
---|
| Suchanfrage | Suchkonsole | Welche Suchanfragen und Seiten sind sichtbar? |
| Sichtbarkeit der KI-Reaktion | Manuelle Prüfungen und KI-Sichtbarkeitstools | Wird die Marke zitiert oder zusammengefasst? |
| KI-Referenzen | Webanalyse und Serverprotokolle | Welche KI-Systeme senden Datenverkehr? |
| Agentensitzungen | Protokolle und Ereignisverfolgung | Lesen, vergleichen oder handeln Agenten? |
| Ausführungsergebnisse | Produktregistrierungen, Warenkörbe, API, Reservierungen oder Bestellungen | War die Aufgabe erfolgreich? |

Der [Leitfaden zur AEO-KPI-Messung](/es/docs/aeo-kpis-measurement/) erläutert das umfassendere KPI-Modell.

## Was Sie für AEO SEO registrieren müssen

Mindestens:

- Abfrage und Zielseite von der Search Console
- Seitentitel und Meta-Beschreibung zum Zeitpunkt der Änderung
- AI-Referenzquelle
- Zugriff auf llms.txt
- Bekannter Tracker oder User-Agent-Agent
- Anrufe an Terminals
- Formular beginnt und endet
- Warenkorb starten und Bezahlvorgang abschließen
- politische Ablehnungen
- Konformitäts- oder Bestätigungsstatus

Um mehr über Handelsabläufe zu erfahren, lesen Sie [Agent Trading Attribution](/es/docs/agentic-commerce-attribution/). Eine breitere Suchebene finden Sie unter [Google AI-Modus in der Search Console](/es/docs/google-ai-mode-search-console-reporting/).

## Verwendung eines aufsteigenden Abfragesignals

Wenn eine Abfrage zunimmt:

1. Identifizieren Sie alle Seiten, die Impressionen erhalten.
2. Wählen Sie die Seite aus, die am besten zur Hauptabsicht passt.
3. Verbessern Sie Titel, Meta-Beschreibung, Einleitung, Überschriften und interne Links.
4. Erstellen Sie Supportseiten nur für verschiedene Unterabsichten.
5. Fügen Sie diese Seiten zu llms.txt hinzu, wenn sie strategisch wichtig sind.
6. Überprüfen Sie die Search Console erneut, nachdem Sie erneut gecrawlt haben.

Der [AEO SEO Implementation Plan](/es/docs/aeo-seo-implementation-plan/) macht daraus einen 30-Tage-Sprint.

## Erwartete Zuordnungslücken

| Lücke | Warum es passiert | Praktische Antwort |
|
---|
---|
---|| No-Click-KI-Reaktion | Der Benutzer erhält genügend Antwort in der Schnittstelle | Impression-Tracking- und Marken-Tracking-Abfragen |
| Fehlender Referrer | Application Strips Quelle oder Assistent | Verwenden Sie Serverprotokolle und Landingpage-Muster |
| Direkter Traffic nach Einsatz von AI | Der Benutzer kommt später zurück | Vergleichen Sie das Abfragewachstum und die Conversion-Pfade |
| Agentenaktion ohne Seitenaufruf | Der Agent verwendet einen Endpunkt oder eine strukturierte Datei | Verfolgung von llms.txt, API und Aktionsprotokollen |

## Häufig gestellte Fragen

### Kann die Search Console KI-Empfehlungen messen?Die Search Console misst die Leistung der Google-Suche, nicht alle KI-Benchmarks. Verwenden Sie es, um Nachfrage und Seitenleistung abzufragen, und kombinieren Sie es dann mit Analysen und Protokollen.

### Was ist die beste AEO-SEO-Metrik?

Es gibt keine einheitliche Metrik. Kombinieren Sie Abfrageimpressionen, CTR, KI-Verweise, Agentenquellenereignisse und Aufgabenergebnisse.

### Soll ich für jede Popup-Abfrage eine Seite erstellen?

Nein. Verbessern Sie zuerst die Seite, die am besten passt. Erstellen Sie neue Seiten nur für unterschiedliche Zwecke.

### Wie entspricht llms.txt den Messungen?

llms.txt hilft Agenten dabei, wichtige Seiten zu finden. Serverprotokolle können angezeigt werden, wenn sie von KI-Systemen oder -Tools angefordert werden.
