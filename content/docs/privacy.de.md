## Datenerfassung

Diese Website ist eine statische Website, die auf GitHub Pages gehostet wird. Es werden standardmäßig keine Cookies, Trackingpixel oder Analyseskripte verwendet. Über diese Website werden keine personenbezogenen Daten erhoben, gespeichert oder verarbeitet.

Weitere Informationen finden Sie in der [GitHub-Datenschutzerklärung](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).
