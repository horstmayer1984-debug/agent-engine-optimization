#Chrome WebMCP オリジン テスト チェックリスト

Chrome の WebMCP ジョブは、Web サイトに構造化ツールをブラウザベースのエージェントに公開する方法を提供します。現実的なチャンスはバズワードとしての「AIナビゲーション」ではありません。フォーム、構成、診断、カート、ワークフローを十分理解しやすくし、エージェントがより少ないクリック数でアクションを実行できるようにすることが重要です。

## WebMCP についての Chrome の意見

Chrome [WebMCP ドキュメント] (https://developer.chrome.com/docs/ai/webmcp) では、WebMCP を AI エージェントに構造化ツールを公開するための Web 標準として提案されていると説明しています。 JavaScript API と HTML フォームの注釈を使用して、エージェントにページ機能の操作方法を知らせることができます。

Chrome の [I/O 2026 開発者アップデート] (https://developer.chrome.com/blog/chrome-at-io26) では、Web サイトをエージェント ツールキットに変換する方法として WebMCP について説明しています。 WebMCP ページでは、ローカル開発フラグとソース テスト パスも参照します。

より広範な概要については、既存のガイド [WebMCP for Agent-Ready Websites](/es/docs/webmcp-agent-ready-websites/) を参照してください。このページでは、チームがソース テストまたはプロトタイプ ツールに参加する前の準備作業に焦点を当てます。

|

---|
|フォーム |フィールド名、ラベル、検証、エラー状態 |エージェントには視覚的な推測ではなく、安定した意味が必要です。 |

|代替のユーザー エクスペリエンス |通常の人の流れはまだ機能します。 WebMCP は段階的に改善されるはずです。 |
|不正行為の規制 |レートキャップ、認証制御、およびポリシー障壁 |構造化ツールは便利な操作を公開できます。 |

## WebMCP の最初の候補として最適

安全で区切られたワークフローから始めます。

1. 製品カタログ内を検索します。
2. 構成ページで診断を実行します。
3. ドラフトリクエストを保存します。
4. 注文状況を確認します。
5. 支援ルートを見つける。
6. プランの機能を比較します。
7. 送信せずにフォームに記入します。

最初は、購入、アカウントの削除、契約の変更、医師のアドバイス、送金などのリスクの高い行動を避けてください。後で公開する場合は、明示的な確認と監査ログを要求してください。

## WebMCP ツールを追加する前に修正すべきこと

1. 「続行」などのあいまいなボタンを特定のラベルに置き換えます。
2. 検証メッセージを対応するフィールドの横に配置します。
3. 重要なコンテンツを HTML で利用できるようにします。
4. 安定した ID または名前をフォーム コントロールに追加します。
5. 成功したアクションの結果を文書化します。
6. エラーの意味とエラーからの回復方法を文書化します。
7. 読み取り専用ツールを書き込みアクションから分離します。これらの同じ修正により、[エージェント UX と AEO SEO](/es/docs/agent-ux-aeo-seo/) および [Lighthouse Agentic Browsing Audit](/es/docs/lighthouse-agentic-browsing-audit/) が改善されます。

|
---|
---|
---|
|エージェントの動作方法 | UI ベースのクリック、タイプ、推測 |構造化されたツール定義を使用する |

これが、WebMCP がエージェントのナビゲーションに重要である実際的な理由です。WebMCP により曖昧さが軽減されます。

## よくある質問

### WebMCP 本番環境の準備はできていますか?

緊急事態として扱います。 Chrome はこれを、普遍的な運用要件ではなく、ローカル開発とソース テスト パスのサポートを備えた提案された標準であると説明しています。

### WebMCP は MCP サーバーを置き換えますか?

いいえ、MCP サーバーはブラウザの外部にツールとコンテキストを公開します。 WebMCP は、構造化ツールをエージェントに公開するブラウザベースの Web サイトです。

### eコマース サイトは WebMCP 経由での支払いを公開する必要がありますか?

最初ではありません。読み取り専用の製品検索、在庫検索、ポリシーの取得、カートの準備を始めましょう。支払いと注文の履行には、より厳格な承認と監査管理が必要です。

### WebMCP は SEO に役立ちますか?

間接的に。クロール可能なコンテンツ、タイトル、説明、構造化データの代替となるものではありません。これは、エージェントがサイト上で動作する必要がある場合に、実行層に役立ちます。
