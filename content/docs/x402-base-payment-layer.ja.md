Base の x402 は、エージェント エンジン最適化のために最も速く、安価で、最も広く採用されている有料レイヤーです。 AI エージェントに HTTP 経由で直接USDC マイクロペイメントを即座に提供し、すべての API 呼び出しまたはコンテンツ リクエストを 1 秒以内の決済で自動収益に変えます。

## Base が x402 トラフィックを支配する理由

Base は、OP スタック上に構築された Coinbase のレイヤー 2 ネットワークです。 2026 年 4 月時点で、x402 トランザクション量の約 75 パーセントを処理します。3 つのプロパティにより、エージェント支払いのデフォルトの選択肢となっています。

速度: トランザクションは 2 秒以内に完了します。エージェントが HTTP 要求と応答のサイクル中に支払いを行う場合、これは、要求がタイムアウトになる前に支払いが検証されることを意味します。

コスト: 手数料は取引ごとに 0.001 ドル未満です。これにより、マイクロペイメントが実現可能になります。取引手数料が 3 桁低い場合、価格 0.01 ドルのデータ クエリは経済的に実行可能です。

インフラストラクチャ: Coinbase 開発者プラットフォームは、ウォレットの作成、ガススポンサーシップ (ブローカーが手数料を支払うために ETH を保持する必要がないように)、およびすぐに使えるコンプライアンスツールを提供します。

## AEO サイトのビジネス上のメリット

リクエストごとにあらゆるエンドポイントを収益化します。プレミアム データ ソース、IT 運用、分析ツール、コンテンツ アクセスの価格は個別に設定されています。サブスクリプション管理、アカウント作成、請求は必要ありません。

エージェント向けのゼロフリクションKYC。資金に余裕のある自営業のエージェントは、即座に支払いを行うことができます。登録、本人確認、利用規約への署名など​​は必要ありません。

チェーンレシートによるリアルタイム決済。各支払いはBaseブロックチェーン上で検証可能です。これにより、エージェントとそのオペレーターとの信頼を築く監査証跡が作成されます。

## 比較: x402 のベース チェーンと他のチェーン

|チェーン |目的 |料金 | x402 の採用 (2026 年 4 月) |
|---|---|---|---|
|ベース | 2 秒未満 | $0.001 未満 |優勢、ボリュームの約 75% |
|ソラナ | 1 秒未満 |ゼロに近い |成長中 |
|イーサリアムメインネット | 12秒 |メジャー、変数 |マイクロペイメントの導入の減少 |

イーサリアムのメインネット手数料により、ほとんどのユースケースではマイクロペイメントは非現実的になります。 Solana は、もう少し高速な目的を備えた実行可能な代替手段です。 Coinbase の統合と x402 Foundation アプローチにより、Base が導入をリードします。

## x402 の Base を始める

Coinbase 開発者プラットフォームまたはサポートされているウォレットプロバイダーを通じて、Base に USDC ウォレットを作成します。 x402 ミドルウェアを更新して、402 応答の本文の文字列として「base」を指定します。 Coinbase のガス スポンサーシップ機能を使用すると、エンドポイントの料金を支払うエージェントが別途 ETH を保有する必要がなくなります。テストには、テスト USDC で Base Sepolia (テストネット) を使用します。ミドルウェアをデプロイし、テスト支払いを実行し、本番環境に移行する前にフロー全体を検証します。

[x402 導入ガイド](/es/docs/implementing-x402/) では、ミドルウェア構成について説明しています。 [x402 概要](/es/docs/x402-agent-payments/) は戦略的背景を説明しています。

## よくある質問

**x402 にイーサリアム メインネットを使用してみてはいかがでしょうか?**
主要ネットワークのガス料金により、少額決済は現実的ではありません。 2 ドルのガス料金を伴う 0.01 ドルのデータ クエリは機能しません。 Base は 0.001 ドル未満の手数料でこの問題を解決します。**エージェントはベースに ETH を持っている必要がありますか?**
必ずしもそうとは限りません。 Coinbase のガス スポンサーシップ機能を使用すると、サイト運営者がガス料金を負担できるため、エージェントの必要がなくなります。

**Baseで受け取ったUSDCを法定通貨に変換するにはどうすればよいですか?**
Coinbase、Base USDCをサポートする取引所、またはステーブルコイン決済を受け入れる支払いプロセッサを通じて。決済は通常 24 時間以内に完了します。

**Base は運用環境での使用に十分な分散化を実現していますか?**
Base は Coinbase によって運営される L2 であり、完全に分散化されたチェーンよりも集中化されたシーケンサーを備えていることを意味します。エージェントによるマイクロペイメントの場合、スピードとコストの利点が分散化のトレードオフを上回ります。 x402 Foundation のマルチチェーン サポートはオプションを提供します。

**ガソリン スポンサーシップとは何ですか?**
サイト運営者がエージェントに代わって少額のブロックチェーン取引手数料を支払う機能。エージェントがネイティブ チェーン トークンを保持する必要がなくなり、摩擦がゼロになります。
